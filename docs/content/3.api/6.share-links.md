---
title: useShareLinks()
description: A composable that generates social share URLs for the current page across multiple platforms.
---

## Usage

Use the auto-imported `useShareLinks` composable to generate share URLs for the current page.

```ts twoslash
const links = useShareLinks()
// ComputedRef<ShareLinks>
```

Access individual platform URLs from the returned computed ref:

```vue
<template>
  <a :href="links.twitter" target="_blank">Share on Twitter</a>
  <a :href="links.facebook" target="_blank">Share on Facebook</a>
</template>
```

### UTM Tracking

Append UTM parameters to all generated share URLs:

```ts twoslash
const links = useShareLinks({
  utm: {
    utm_source: 'website',
    utm_medium: 'social',
    utm_campaign: 'spring-launch',
  },
})
```

### Custom URL and Title

Override the default canonical URL and title:

```ts twoslash
const links = useShareLinks({
  url: 'https://example.com/special-page',
  title: 'Check out this page',
  platforms: ['twitter', 'facebook', 'email'],
})
```

## Options

### `url`

- Type: `MaybeRefOrGetter<string | undefined>`{lang="ts"}
- Default: canonical URL of the current page

Override the URL to share. Defaults to the canonical URL resolved from the current route and site config.

### `title`

- Type: `MaybeRefOrGetter<string | undefined>`{lang="ts"}
- Default: site name from site config

The title or text included in the share link. Falls back to the `name` from your site config.

### `platforms`

- Type: `SharePlatform[]`{lang="ts"}
- Default: `['twitter', 'facebook', 'linkedin', 'whatsapp', 'telegram', 'reddit', 'pinterest', 'email']`{lang="ts"}

Which platforms to generate share links for. Restricting this avoids generating unused URLs.

### `utm`

- Type: `MaybeRefOrGetter<ShareLinkUtmParams | undefined>`{lang="ts"}
- Default: `undefined`{lang="ts"}

UTM parameters appended to the shared URL. Supports `utm_source`, `utm_medium`, `utm_campaign`, `utm_term`, and `utm_content`.

## Supported Platforms

| Platform    | Share behaviour                        |
| ----------- | -------------------------------------- |
| `twitter`   | Tweet with URL and text                |
| `facebook`  | Facebook share dialog                  |
| `linkedin`  | LinkedIn share offsite                 |
| `whatsapp`  | WhatsApp message with title and URL    |
| `telegram`  | Telegram share with text               |
| `reddit`    | Reddit submit with URL and title       |
| `pinterest` | Pinterest pin with URL and description |
| `email`     | `mailto:` link with subject and body   |

## Return Value

- Type: `ComputedRef<ShareLinks>`{lang="ts"}

Where `ShareLinks` is `Record<SharePlatform, string>`, a map of platform name to fully encoded share URL.
